DIY vpn Server

Published on 2020-06-01

I'm using vultr as the provider for the vpn server host. (But it's possible to choose any VPS provider) Bellow are the steps I follow to create the vpn server.

  1. Build a virtual private server.(I'm using 3.50 USD/month configuration)
  2. Install ubuntu 18.04 on it.
  3. Once installation is finished ssh in to the server.
  4. Clonealgo to the home directory in the server.
  5. Follow the installtion guide on repository.
  6. Once installation is done copy the config files to all the devices.

Monthly VPN bandwidth capped at host configuration bandwidth devided by two. (In my configuration it's 500GB/2 = 250GB).
