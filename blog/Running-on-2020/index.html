Running on 2020

Published on 2020-12-17

Yesterday I stopped running for few months. This is me bragging about my running on 2020.

I started this running season somewhere on April. Mostly because there isn't anything else to do (stuck at home due to covid restrictions).

Around May I came across the book "Running Formula" by "Jack Daniels" and started practicing + tracking runs. By the June I did 5k. I stopped the practicing by the book once I made it to a certain level. (There were bunch of ranking levels in the book)

And started running for just fun.

Then came the virtual runs. Idea of virtual run is you sign up for a certain distance to do within a certain time period.

I did few of them.

  • SMU SwimRunSprint(no swimming this year)

  • SMOO Challenge

  • Straits Times virtual run 17.5km

  • Straits Times virtual run 175km

Longest run I did is a half marathon(which took almost 3 hours). With a pretty bad pace.

Now it's time for go back to book again and improve the pace(after few months rest)
