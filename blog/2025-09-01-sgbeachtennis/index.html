![full-width](/assets/blog/sgbeachtennis.png) New website for the Singapore Beach Tennis is up at [https://sgbeachtennis.com/](https://sgbeachtennis.com/) For a while I wanted to create a website for beachtennis. To promote the sport and as a give back to the community. To create the site I used Hugo, and hosted it on cloudflare pages. Domain is bought from porkbun. For the initial version I wrote a "wiki" page and "about" page. I'm also planning to run an interview series with the players.
