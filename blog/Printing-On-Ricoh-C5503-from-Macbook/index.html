Printing On Ricoh C5503 from Macbook

Published on 2020-12-22

I have access to a Ricoh 5503 printer. It works fine with windows. It used to work fine with Linux. But didn't work with Mac. Then it stopped working on Linux(After updating to 20.04)

Since I didn't have a windows PC, I set upon to make it work on Mac.(I'm using a office issued macbook primarily for work) After some googling I found this posthttps://community.spiceworks.com/topic/1060351-having-issues-printing-off-of-a-ricoh-mp-c5503-and-ricoh-mp-c3003

I followed the (relevant to big sur anyways) steps and it worked!!!

What I did exactly is,

  1. Key in the ip address in Address field.
  2. Set the Protocol to IPP
  3. Set the Use to Generic PCL Printer.
  4. Add the printer.

It asked me whether I want to use it as a Duplexer and I said yes.

That's all.
