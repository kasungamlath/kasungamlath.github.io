My First Contribution to Firefox

Published on 2020-05-21

My fist code contribution to Mozilla Firefox got accepted. It's just fixing a code analysis bug.(and fixing another code analysis warning which caused by the fix). Anyways it feels good. I have being using the Firefox since I started using a PC. Since somewhere around 2007. So It's good to finally give back.

There are multiple guides on how to make the first contribution. This is what I followed https://firefox-source-docs.mozilla.org/contributing/how_to_contribute_firefox.html. To find the bug I used https://codetribute.mozilla.org/.

This is the issue on Bugzilla https://bugzilla.mozilla.org/show_bug.cgi?id=1626754. This is the issue on Phabricator https://phabricator.services.mozilla.com/D75704.

It's scheduled to go live with version 78 !!!
